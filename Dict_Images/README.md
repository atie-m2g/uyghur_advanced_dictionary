# چۈشەندۈرمە
گېت ئامبىرى يوللاش ھۆججەت چەكلىمىسى 25 مىگا دىن تۆۋەن ، شۇڭا لۇغەت رەسىمىنى پارچىلاپ چىقاردۇق .

بېسىلما ھۆججەتلەر (zip files) ئۇيغۇر - ئۇيغۇر لۇغىتى تەسۋىرلەنگەن ھۆججەتلەر . ئىشلىتىش ئۈچۈن بارلىق بېسىلما ھۆججەتلەرنى شىشىپ (unzip) بىر ھۆججەت قىلىنسا بىر ھەققى پۈتۈن لۇغەت بولىدۇ  . بارلىق بېسىلما ھۆججەتلەر مۇستەقىل ھۆججەت بولۇپ ، 7zip بېسىلما ھۆججەتلەرنى شېشىنى  قوللايدىغان ھەر قانداق شىش (unzip)  قوراللىرى بىلەن شىشىپ قىلغىلى بولىدۇ .

# Intro
The GitHub upload file limit is less than 25MB, so make it split to upload.

The ziped files are Uyghur - Uyghur dictionary imaged files. For use just extract all zip to make one file. The all zip file is independent file, can extract with any unzip tools which support 7zip pressed ones.
